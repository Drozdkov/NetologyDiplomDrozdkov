# Дипломный проект профессии «1C-программист: с нуля до middle»

## Инструкция по первичной настройке обновленной конфигурации "Управление ITФирмой" (версия 1.0.2.0)

### Настройка ролей пользователей

#### Настройка профилей групп доступа

*Администрирование - Настройка пользователей и прав - Профили групп доступа*

Создайте следующие профили:
- БухгалтерИТФирмы (базовая функциональность + роли "ВКМ_Массовое Создание Документов "Реализация Товаров и Услуг", "ВКМ_Работа с документом "Обслуживание клиентов", "ВКМ_Работа с отчетом "Анализ выставленных актов"),
- КадровикРасчетчик (базовая функциональность + роли "ВКМ_Ввод данных об окладах и премиях", "ВКМ_Ввод данных об отпусках", "ВКМ_Заполнение данных графика", "ВКМ_Начисление и выплата зарплаты", "ВКМ_Работа с отчетами по зарплате и отпускам", "ВКМ_Работа с отчетом "Выработка специалистов"), 
- Менеджер (базовая функциональность + роли "ВКМ_Работа с документом "Обслуживание клиентов", "ВКМ_Работа с параметрами рассылки в телеграм"),
- Специалист (базовая функциональность + роли "ВКМ_Работа с документом "Обслуживание клиентов", "ВКМ_Работа с отчетом "Выработка специалистов", "ВКМ_Работа со справочником "Виды работ").

Рабочие задачи, соответствующие профилям групп доступа:
- Менеджер: заполнение справочника "Виды работ" для работы с документом "Обслуживание клиента". Создание документов "Обслуживание клиентов". Настройка параметров подключения бота в Телеграм для рассылок уведомлений специалистам;
- Специалист: Получение уведомлений в Телеграм. Прикрепление сканов актов выполненных работ к документу "Обслуживание клиента". Просмотр отчета "Выработка специалистов";
- БухгалтерИТФирмы: массовое создание документов "Реализация товаров и услуг" на основании документов "Обслуживание клиента", введенных в предыдущем периоде. Работа с отчетом "Анализ выставленных актов".
- КадровикРасчетчик: Заполнение графика работы, внесение данных об условиях оплаты сотрудников, создание и проведение документов "Начисление заработной платы", "Начисление фиксированной премии", "Выплата зарплаты", "График отпусков". Анализ графика отпусков на год на диаграмме Ганта. Работа с отчетами "Выработка специалистов", "Расход запланированных отпусков" и "Расчеты с сотрудниками".

### Настройка параметров уведомления в Telegram

*Обслуживание клиентов - Сервис - Идентификатор группы оповещения в Телеграм (константа)*
*Обслуживание клиаентов - Сервис - Токен управления Телеграм-ботом (константа)*

1. Создайте группу и бота для рассылок уведомлений в соответствии с [описанием реализации](telegram.md)
2. Внесите параметры подключения к созданному боту в соответствующие константы. 
------

### Заполнение значений констант

*Обслуживание клиентов - Сервис - Номенклатура абонентская плата (константа)*
1. Выберите из списка и запишите значение константы "Абонентская плата по договору"

*Обслуживание клиентов - Сервис - Номенклатура выполненные работы (константа)*
1. Выберите из списка и запишите значение константы "Дополнительные работы по договору абонентского обслуживания"
------

### Внесите данные о категориях сотрудников, их окладах и премии (процент от выполненных работ)

*Начисление зарплаты - Условия Оплаты Сотрудников*

-  Внесите информацию по всем сотрудникам.
-  Сотрудники категории "Административный персонал" получают только оклад. Для них не обязательно заполнять поле "Процент от работ".
-  Сотрудники категории "Специалист" получают оклад и премию (процент от выполненных работ). Для них необходимо заполнять поле "Процент от работ". При этом значение поля может быть нулевым.

------

### Заполнение графика работы

*Начисление зарплаты - Сервис - Заполнение графика*
Выберите месяц для заполнения и нажмите на команду "Заполнить". Просмотреть результат заполнения можно в *Начисление зарплаты - Сервис - График работы*
